# FF Resource Dashboard

FF Resource Dashboard 是一个面向游戏资源投放数据的可视化分析看板，用于分析 2024-2026 年不同资源品质、活动节点和斋月周期下的资源投放趋势。

在线访问：<https://dxawdc.github.io/FF-resource-dashboard/>

## 功能详情

### 1. 数据加载

- 默认读取项目根目录下的 `data.csv`。
- 支持输入 GitHub Raw CSV 链接进行在线拉取。
- 支持本地上传 CSV 文件并即时重绘图表。
- 当本地环境无法读取 `data.csv` 时，会自动回退到内置模拟数据。

### 2. 全局筛选

看板顶部提供多维筛选能力，所有图表会同步响应：

- 统计指标切换：
  - 按资源个数（数量）
  - 按总价值（数量 × 价值）
- 活动类型筛选
- 资源类型筛选
- 价值高低筛选

### 3. 资源品质投放趋势分析

模块一展示不同资源品质在多个时间粒度下的投放趋势：

- 年度各品质总投放量对比
- 季度各品质总投放量对比
- 月度投放量与品质结构变化
- 节点各品质总投放量对比

这些图表均使用堆叠柱状图展示不同品质资源的构成，并在柱顶显示当前可见系列的合计值。

### 4. 斋月周期分析

模块二用于观察斋月前、中、后资源投放的变化：

- 斋月等长周期资源品质波动对比
- 斋月特定时段资源品质波动对比

支持按年份和阶段对比斋月周期内外的资源投放结构。

### 5. 图表交互

- 点击单个堆叠柱区块：查看该时间段下对应资源品质的明细数据。
- 点击柱顶透明总计区域或总计数字：查看该时间段下全部品类的汇总与明细。
- 点击图例隐藏或显示某个品质时：柱顶合计数字会实时重新计算，只统计当前可见的堆叠项。
- 图表 tooltip 会根据当前统计指标展示数量或总价值。

### 6. 数据明细弹窗

点击图表后会打开明细弹窗，包含：

- 当前筛选条件说明
- 所选数据记录总数
- 资源价格计算汇总表
- 底层原始数据明细表
- 汇总 CSV 一键导出

汇总表会按资源名称、活动类型、资源类型、资源品质进行聚合，并计算投放总数量与总价值合计。

## 技术栈

- HTML5：页面结构
- Tailwind CSS CDN：样式与响应式布局
- Apache ECharts 5.5：堆叠柱状图与交互式可视化
- PapaParse 5.4：CSV 文件解析与远程 CSV 拉取
- Vanilla JavaScript：数据处理、筛选联动、图表渲染与弹窗交互
- GitHub Pages：静态网页托管

## 数据格式

项目默认读取 `data.csv`，主要依赖以下字段：

- `数量`
- `价值`
- `活动上线日期`
- `资源品质`
- `价值高低`
- `资源类型`
- `活动类型`
- `节点`
- `资源名称`

其中：

- 投放数量来自 `数量` 字段。
- 总价值 = `数量 × 价值`。
- 年、季度、月份会根据 `活动上线日期` 自动解析。
- 空分类、未知分类和未分类数据会在处理阶段被过滤。

## 本地运行

可以使用任意静态文件服务器运行，例如：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000/index.html
```
